# 08 — 测试体系（Testing）

**核心内容**：pytest、Test Doubles、分层测试（Web / Service / DAO）、集成与 CI、负载与安全测试选型。  
**核心目标**：在 FastAPI 项目里搭**从单元到上线前验证**的完整闭环。

对照代码：可在 `fastapi_app/` 上逐步补充 `tests/`，与官方 [Testing](https://fastapi.tiangolo.com/tutorial/testing/) 一致。
